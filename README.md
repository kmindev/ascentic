# ascentic (향 제품 구독/판매 쇼핑몰)

## 프로젝트 소개
요즘 시중에 너무 많은 향이 존재하는데, 사용자의 취향을 찾고 직접 체험하며
향과 관련된 다양한 제품들을 구독 · 판매하는 웹 사이트

## 개발 기간
2023.04 ~ 2023.06 (8주)


## 기술
### Front-End
React.js

### Back-End
Spring Boot(JPA)

### DB
MySQL

### Open API
- kakao 로그인 API
- Toss Payments 결제 API
